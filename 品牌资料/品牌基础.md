# 品牌基础

## 品牌基本信息

| 项目 | 内容 |
|------|------|
| 品牌全称 | 上海尚海建筑装饰工程有限公司（简称"尚海整装"） |
| 品牌定位 | 聚通装饰集团旗下整装战略品牌，定位为"第八代个性化透明整装品牌" |
| 品牌口号 | 好装修，在尚海 |
| 品牌使命 | 创意幸福人居，享受快乐装潢 |
| 核心价值观 | 聚人心，通天下——聚客户之心路通天下、聚员工之心业通天下、聚合作伙伴之心情通天下、聚社会之心誉通天下 |

## 品牌发展历程

- **聚通装饰集团**创立于1994年（工商注册1995年1月28日），至今已深耕上海家装行业**32年**
- **尚海整装**：2013年成立"尚海装饰"；2018年正式转型为"尚海整装"
- **创始人**：徐国俭，聚通装饰集团董事长，中国室内装饰协会副会长、上海市室内装饰行业协会会长
- **装修模式迭代**：第一代传统装修 → 第二代清包工+辅材 → 第三代半包 → 第四代半包+主材 → 第五代套餐 → 第六代样板间 → 第七代个性化整装买手 → **第八代个性化透明新整装（2025年12月发布）**
