# 关镇铨整装

- **成立时间**：1996年
- **总部**：上海
- **创始人**：美籍华人关镇铨
- **定位**："高品质+高性价比"，一站式家居解决方案
- **理念**："好设计+好工程+好材料+好价格"
- **荣誉**：服务超10万户上海家庭
